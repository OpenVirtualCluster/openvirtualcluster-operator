OpenVirtualCluster Helm Charts

Welcome to the OpenVirtualCluster Helm Chart repository. This repository hosts Helm charts for deploying OpenVirtualCluster components in Kubernetes clusters.

Available Charts

openvirtualcluster-operator

The OpenVirtualCluster Operator manages virtual Kubernetes clusters within a host Kubernetes cluster.

This chart deploys the operator that watches VirtualCluster custom resources and manages their lifecycle.

Installation:

helm repo add openvirtualcluster https://openvirtualcluster.github.io/openvirtualcluster-operator/
helm repo update
helm install openvirtualcluster openvirtualcluster/openvirtualcluster-operator

Usage

Add the Helm repository to your Helm client:

helm repo add openvirtualcluster https://openvirtualcluster.github.io/openvirtualcluster-operator/
helm repo update

List the charts in the repository:

helm search repo openvirtualcluster
